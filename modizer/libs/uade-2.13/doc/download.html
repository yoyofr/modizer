Git version control server

Version control server:

To sync UADE sources, use Git:

git clone git://zakalwe.fi/uade uade.git

The old CVS server is not in use anymore. You can download the full CVS database from this location. The whole version history was imported to the Git repository, so it is not really needed anymore.

 
Source code based distributions:

FreeBSD:
Martin Blabb maintains UADE ports for FreeBSD.

Gentoo Linux:
The latest version of UADE can usually be found from the portage tree. You should emerge app-emulation/uade.

RSYNC server for Amiga chip songs
A public rsync server for various Amiga chip songs (over 8000 files or about 300 MiBs of data):

rsync -avz zakalwe.fi::chip /local/path/

The value of rsync protocol is easy and fast updates of the collection. The tool transfers only new or changed bits. Therefore bandwidth is not wasted, and updates are fast.
