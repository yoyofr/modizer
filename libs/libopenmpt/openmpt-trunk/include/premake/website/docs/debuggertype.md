debuggertype

```lua
debuggertype "value"
```

### Parameters ###

`value` one of:
* `Mixed` - needs documentation.
* `NativeOnly` - needs documentation.
* `ManagedOnly` - needs documentation.

### Applies To ###

The `config` scope.

### Availability ###

Premake 5.0.0 alpha 12 or later.

### Examples ###

```lua
debuggertype "value"
```
