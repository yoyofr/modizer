LIBKSS VM Information

# I/O MAP

```
06H W GG stereo
40H W EXTRA I/O SELECT (Write 07FH to activate I/O 41H-44H)
41H W STOP REQUEST (Write 01H to stop current playback)
42H W LOOP COUNTER
43H W MGSDRV @m data address (L)
44H W MGSDRV @m data address (H)
7CH W MSX-MUSIC ADDRESS
7DH W MSX-MUSIC DATA
7EH W SN76489 (mirror)
7FH W SN76489
A0H W PSG ADDRESS
A1H W PSG DATA WRITE
A2H R PSG DATA READ
AAH W 1bit D/A (Bit7)
ABH W 1bit D/A (Bit1)
C0H W MSX-AUDIO ADDRESS
C1H W MSX-AUDIO DATA WRITE
C2H R MSX-AUDIO DATA READ
F0H W FMUNIT ADDRESS
F1H W FMUNIT DATA
FEH W KSS16KB bank select port
```

# MEMORY MAP
## KSS16K

```
0000-3FFFH RAM
4000-7FFFH RAM
8000-BFFFH 16KB Banked ROM, SCC(9800-988FH/B800-B8AFH)
C000-FFFFH RAM
```

## KSS16K/RAM

```
0000-3FFFH RAM
4000-7FFFH RAM
8000-BFFFH RAM or 16KB Banked ROM, SCC is disabled.
C000-FFFFH RAM
```

## KSS8K

```
0000-3FFFH RAM
4000-7FFFH RAM
8000-9FFFH 8KB banked ROM (Selector:9000H), SCC(9800-988FH)
A000-BFFFH 8KB banked ROM (Selector:B000H), SCC(B800-B8AFH)
C000-FFFFH RAM
```

## Konami Majong2 (Majutusi)
```
5000-5FFFH 8bit D/A port.  
```
